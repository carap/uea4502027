Exploratory Data Analysis: 911 Calls (Solutions)¶

Complete the instructions ($I$) or answer the questions ($Q$) using your Python3 kernel and your Data Science skills!

For this EDA we will be analyzing some 911 call data from Kaggle. Please, download de compressed data file in this link, and then extract its content. The 'data.csv' file contains the following fields:

  • lat : String variable, Latitude
  • lng: String variable, Longitude
  • desc: String variable, Description of the Emergency Call
  • zip: String variable, Zipcode
  • title: String variable, Title
  • timeStamp: String variable, YYYY-MM-DD HH:MM:SS
  • twp: String variable, Township
  • addr: String variable, Address
  • e: String variable, Dummy variable (always 1)
